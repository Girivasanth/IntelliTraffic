# IntelliTraffic
Real-Time Vehicle Detection: YOLO - based object recognition measures lane-wise density for two wheelers, cars, trucks, and heavy vehicles.
Reinforcement Learning in SUMO learns traffic flow patterns and predicts future congestion to optimize signals dynamically.
The Priority-Based Constraint Model uses Google OR-Tools; it ensures fair green-time allocation, sets minimum and maximum thresholds, and prioritizes emergency vehicles.
Decentralized Edge Controllers provide local resilience with cloud-backed monitoring and coordination across the city.
